# Web Desktop

一个基于 Vue 3 的 Web 桌面环境，模拟 Windows 10 操作体验。

[演示 Demo](https://zijid.github.io/web-desktop/)

## 功能

- 文件管理器（类 Windows 资源管理器）
- 记事本 / 代码编辑器 / 浏览器
- 新软件制作工具（图形化创建应用）
- 系统设置（壁纸、扩展名关联、桌面路径、启动设置）
- 开机动画（可自定义耗时、显示加载日志）
- 右键菜单、文件拖放
- 多图片壁纸、多种背景显示模式
- IndexedDB 文件存储

## 开发

`ash
npm install
npm run dev
`
